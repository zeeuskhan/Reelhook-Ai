## 🌟 Why ReelHook AI?

In the fast-paced world of short-form vertical video, the first **3 seconds** decide whether your audience is going to watch till the end or keep scrolling. **ReelHook AI** analyzes retention psychology to formulate dynamic opening lines ("Pattern Interrupts") that boost average watch times and trigger platform algorithms for viral distribution.

Created using **Google AI Studio** and powered by the advanced language models of **Gemini AI**, this tool offers unprecedented precision in target niche adaptation and local cultural communication—supporting English, Hindi, Hinglish (our specialty!), Spanish, and French.

## 💻 Running ReelHook AI Locally

Get a local instance up and running on your machine in under three minutes.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your workstation.

### Step 1: Clone and Enter Directory

```bash
git clone https://github.com/zeeuskhan/Reelhook-Ai.git
cd Reelhook-Ai
```

### Step 2: Install Base Dependencies

```bash
npm install
```

### Step 3: Setup Gemini API Key Credentials

To allow the AI features to communicate with Google systems, create a `.env` configuration file in the project root:

```env
GEMINI_API_KEY=your_actual_google_gemini_api_key_here
```

> **Security Note:** Never commit your `.env` secrets file containing active API keys to public repositories like GitHub!

### Step 4: Launch local Dev Server

```bash
npm run dev
```

Your browser will automatically launch the interactive preview at:
**http://localhost:3000**
