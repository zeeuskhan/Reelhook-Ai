## 💻 Run AI Studio App Locally

### Requirements

Node.js installed


### Install dependencies

```bash
npm install
```


### Setup Gemini API Key

Create `.env` file (or use existing variables):

```env
GEMINI_API_KEY=your_api_key
```


### Start development server

```bash
npm run dev
```
